# arpgate

Requirements:

Raspberry Pi 2 Model B
8Gb micro SD card

Installation of Ubuntu Trusty Tahr 14.04 on Rasspberry PI: https://wiki.ubuntu.com/ARM/RaspberryP

Install Puppet: http://stdout.no/puppet-on-raspberry-pi
